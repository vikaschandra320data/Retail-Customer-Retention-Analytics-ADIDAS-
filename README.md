# Retail-Customer-Retention-Analytics-ADIDAS

# Overview
This report summarizes key insights and strategic recommendations from Adidas’ retail customer retention analysis. Using Power BI dashboards, the analysis connects demographics, transactions, loyalty, and churn data to identify customer value drivers and retention opportunities. 

# Tools Used
Power BI, DAX, Dashboard, Data Visualization

# Key Insights
1.	Churn is concentrated among loyal customer with high income, particularly making online purchase.
2.	Customer in the age group of 25-34 tends to purchase more. 
3.	Footwears category is purchased more by loyal customer, also customers prefer to buy from franchise. 
4.	Only half of the loyalty members redeem earned points -engagement opportunity. 
5.	Customers in base tier shows higher CLV and lower churn.

# Source
data/C3_Project_Adidas Retail Customer Retention Analytics.docx.pdf

# Conclusion
The analysis shows that customer retention is most influenced by loyalty participation, purchase frequency, and franchise experience. By strengthening loyalty engagement, targeting at-risk segments with personalized promotions, and standardizing the franchise experience across channels, Adidas can significantly increase Customer Lifetime Value (CLV) and reduce churn. Implementing predictive analytics and ongoing monitoring will ensure long-term retention success.
